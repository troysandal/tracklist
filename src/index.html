Traktor or RekordBox To Human Readable Playlist

You can use this tool to turn a Traktor NML or RekordBox XML file into a human readable format that is ready to post to MixCloud, SoundCloud or wherever you want to give people the list of tracks in your set. Use the Format String to modify what each entry in the playlist looks like. There are multiple fields you can choose from to get the look you want.

Step 3 - Filters

No, we aren't talking about echo, reverb or delay here. we're talking about filtering out tracks you don't want. For example, let's say you previewed 5 tracks before you started, you don't want those showing up. When that's the case set the Starting Track Index to 6 to make the playlist start at the 6th track. If you're using Traktor you can check the Only Show Tracks That Went Live button and it will remove any tracks you previewed but didn't bring up the levels on. Neat eh?
